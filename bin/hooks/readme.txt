// Guide to Adding .git hooks to your local video-player repo

1. Copy/Cut the "hooks.rar" located in the 'bin/hooks' directory of your local repo
2. Paste "hooks.rar" into the '.git/hooks' directory within local your repo
    [".git" folder may be a hidden file]
3. Extract "hooks.rar" here
4. 4 new files should appear without file extensions
    <"post-checkout", "post-merge", "pre-commit", "pre-push"> files
5. For Unix users, ensure this root location's files have executable permissions
